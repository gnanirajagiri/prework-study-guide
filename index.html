HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Using the img tag we can link our images onto our webpage
  • we can add the source and alt attributes to mention the image source and image details to our code
  • The title of the webpage goes uder the title tag in the head section
  • we have a header section and footer section at the top and bottom positions respectively
  • we can even divide our code using section tags on the webpage

CSS

  • CSS contributes the main styles of the webpage
  • using a CSS styles file we can apply some background colours, borders, and font colors to our webpage

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: makes all the updated file to get ready to commit
  • git commit -m "(message>)": commits the files to the repository with a custom message
  • git push: updates all the commited files on to the repository
  • git pull : Pulls the current working condition of the repository

JavaScript

  • JavaScript is the main functioning of the webpage
  • The code on a JavaScript file is responsible fot how the wesite functions
  • The if statements of the JavaScript is a logic statements which executes the code only if the particular contions are met
  • we have for loop to control the flow of the code
  • we can even define functions and call the function to execute a particular part of the code and make the webpage function
